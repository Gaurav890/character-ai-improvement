A single-page report proposing a user-controlled memory/canon layer for Character.AI — what users are already saying about memory in public reviews and Reddit, what Character.AI has already shipped (Story Memory, Facts, Lorebook, etc.), the gaps still open, and five concrete suggestions with a concept mockup.

Public sources only. No fabricated data. Quotes anonymized.

- What's already shipped
- Gaps
- Suggestions
